### Solution to 
```
Сложить два числа


Даны два связанных списка, представляющих два числа (целые числа). Цифры хранятся в обратном порядке, и каждый из узлов содержит одну цифру.
Сложите два числа и верните сумму в виде связанного списка.

Пример 1:

Ввод: l1 = [2,4,3], l2 = [5,6,4]
Выход: [7,0,8]
Пояснение: 342 + 465 = 807.

Пример 2:
Ввод: l1 = [0], l2 = [0]
Вывод: [0]

Пример 3:
Ввод: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Выход: [8,9,9,9,0,0,0,1]
```
